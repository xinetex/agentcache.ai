Quick Start

Get started in 30 seconds with our Python SDK.

Install

pip install agentcache

Use

import agentcache

// Drop-in replacement for OpenAI
const response = agentcache.completion({
  model: "gpt-4",
  messages: [{ role: "user", content: "Hello" }],
  provider: "openai"
});

if (response.get('hit')) {
  console.log("ðŸ’š Cache hit!", response.response);
} else {
  console.log("Cache miss");
}

Python SDK

Official Python client for AgentCache.

Streaming

stream = agentcache.completion(
  model="gpt-4",
  messages=[{ "role": "user", "content": "Write a poem" }],
  stream=True
)

if stream:
    for chunk in stream:
        print(chunk['choices'][0]['delta'].get('content', ''), end="")

Streaming Support

Server-Sent Events (SSE) for cached responses.

curl -X POST https://agentcache.ai/api/cache/get \
  -H "X-API-Key: ac_demo_test123" \
  -d '{
    "provider": "openai",
    "model": "gpt-4",
    "messages": [{"role": "user", "content": "Hello"}],
    "stream": true
  }'

Returns OpenAI-compatible SSE events.

REST API

Direct HTTP API for any language.

POST /api/cache/get

curl -X POST https://agentcache.ai/api/cache/get \
  -H "X-API-Key: ac_demo_test123" \
  -H "Content-Type: application/json" \
  -d '{
    "provider": "openai",
    "model": "gpt-4",
    "messages": [{"role": "user", "content": "Hello"}]
  }'

POST /api/cache/set

curl -X POST https://agentcache.ai/api/cache/set \
  -H "X-API-Key: ac_demo_test123" \
  -d '{
    "provider": "openai",
    "model": "gpt-4",
    "messages": [...],
    "response": "LLM response"
  }'

Swarm API

Execute multi-model swarms with observability.

POST /api/swarm

curl -X POST https://agentcache.ai/api/swarm \
  -H "X-API-Key: ac_demo_test123" \
  -d '{
    "strategy": "fastest",
    "models": [
      {"provider": "openai", "model": "gpt-4"},
      {"provider": "anthropic", "model": "claude-3-opus"}
    ],
    "messages": [{"role": "user", "content": "Hello"}]
  }'

Elastic Overflow

Use AgentCache as a fallback for Redis, ElastiCache, or Vector DBs.

POST /api/overflow

Partner API for revenue-sharing integrations.

// Node.js Partner SDK
const overflow = new AgentCacheOverflow({
  partnerId: 'redis-labs',
  apiKey: '...'
});

  customerId: 'cust_123',
  request: { provider: 'openai', ... }
});

Namespaces
