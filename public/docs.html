Documentation

Complete guide to integrating AgentCache.ai into your AI applications.

Quick Start

Get started in 30 seconds with our NPM package or REST API.

Install

npm install agentcache-client

Use

import { AgentCache } from 'agentcache-client';

const cache = new AgentCache('ac_demo_test123');

// Check cache before calling LLM
const result = await cache.get({
  provider: 'openai',
  model: 'gpt-4',
  messages: [{ role: 'user', content: 'Hello' }]
});

if (result.hit) {
  console.log('✅ Cache hit!', result.latency_ms + 'ms');
  return result.response; // Instant!
}

// Cache miss - call your LLM
const llmResponse = await openai.chat.completions.create({...});

// Store for next time
await cache.set({
  provider: 'openai',
  model: 'gpt-4',
  messages: [{ role: 'user', content: 'Hello' }],
  response: llmResponse.choices[0].message.content
});

NPM Package

TypeScript SDK with full type safety.

Installation

npm install agentcache-client

Methods

cache.get(options)

Check if response is cached

const result = await cache.get({
  provider: 'openai',
  model: 'gpt-4',
  messages: [...]
});

cache.set(options)

Store LLM response in cache

await cache.set({
  provider: 'openai',
  model: 'gpt-4',
  messages: [...],
  response: 'LLM response'
});

cache.moonshot(options)

Call Moonshot AI with reasoning caching

const result = await cache.moonshot({
  model: 'moonshot-v1-128k',
  messages: [...],
  cache_reasoning: true
});

REST API

Direct HTTP API for any language.

POST /api/cache/get

curl -X POST https://agentcache.ai/api/cache/get \
  -H "X-API-Key: ac_demo_test123" \
  -H "Content-Type: application/json" \
  -d '{
    "provider": "openai",
    "model": "gpt-4",
    "messages": [{"role": "user", "content": "Hello"}]
  }'

POST /api/cache/set

curl -X POST https://agentcache.ai/api/cache/set \
  -H "X-API-Key: ac_demo_test123" \
  -d '{
    "provider": "openai",
    "model": "gpt-4",
    "messages": [...],
    "response": "LLM response"
  }'

Namespaces

Webhooks

  • reasoning.reused - Reasoning reused
