FAQ

How do you avoid stale cache entries?

Keys include provider, model, and parameters in the hash. TTLs can be set per request (default 7 days). Tag-based purging coming Q1 2025.

Is my data secure?

All entries are encrypted at rest by Upstash Redis (TLS in transit, AES-256 at rest). PII scrubbing and customer-managed keys coming Q2 2025.

Which providers do you support?

Any LLM provider - OpenAI, Anthropic, Claude tested. Moonshot AI (Kimi K2) has dedicated reasoning caching. Bring your own API keys.

What about function calling and tools?

Include tools/functions in your messages array - they'll be part of the cache key. Custom key material for RAG coming Q1 2025.
