FAQ

How do you avoid stale cache entries?

We version keys with provider model IDs and parameters. TTLs can be set per-call, per-tag, or globally. You can also purge by tag or key via the API.

Is my data secure?

All entries are encrypted at rest with server-side keys, with optional customer-managed keys. PII scrubbing is available for sensitive workloads.

Which providers do you support?

Any HTTP-based LLM API. First-class SDK helpers for OpenAI, Anthropic, and Claude models, with generic adapters for others.

What about tools and function calls?

Tools/functions are part of the cache key. You can also attach custom key material for RAG inputs or retrieval ids.
